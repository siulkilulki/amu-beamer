\section{Options}

\begin{frame}
	\frametitle{Available options}
    AMUBeamer theme supports the following options with the 
    \texttt{\\usetheme[]\{AMUBeamer\}} command:
		\begin{description}
 			\item [\texttt{english}] sets the Adam Mickiewicz University's wordmark on the title page to the English version
 			\item [\texttt{wmi}] sets the colour of decorative bars to WMI red
 			\item [\texttt{wn}] sets the colour of decorative bars to WN violet
 			\item [\texttt{wb}] sets the colour of decorative bars to WB green
 			\item [etc.] \ldots
 		\end{description}
\end{frame}
